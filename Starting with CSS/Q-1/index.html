Inline css is easy way to add css, this css don't need any class and id.

Internal css is second type of using css, where we use class and id to select a particular element. We use this tag inside the head.

External css is third type of using css, where we use class and id to select a particular element. We use this tag out side of our html tag, and creat a perticular file wich is style.css, but to make this process add a link html with css use link tag inside the head.
